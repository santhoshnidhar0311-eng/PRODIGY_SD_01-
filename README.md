# PRODIGY_SD_01 - Temperature Conversion Program

## Task Description

Develop a program that converts temperatures between Celsius, Fahrenheit, and Kelvin scales. The program accepts a temperature value and its original unit as input, then converts and displays the equivalent values in the other two temperature scales.

## Features

* Convert Celsius to Fahrenheit and Kelvin
* Convert Fahrenheit to Celsius and Kelvin
* Convert Kelvin to Celsius and Fahrenheit
* User-friendly input and output
* Accurate temperature conversion formulas

## Technologies Used

* Python

## Conversion Formulas

### Celsius to Fahrenheit

F = (C × 9/5) + 32

### Celsius to Kelvin

K = C + 273.15

### Fahrenheit to Celsius

C = (F - 32) × 5/9

### Kelvin to Celsius

C = K - 273.15

## Example

Input:
Temperature: 25
Unit: Celsius

Output:
Fahrenheit: 77.0 °F
Kelvin: 298.15 K
